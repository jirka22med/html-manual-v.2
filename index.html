Co je to CSS?

Ahoj Jiřík! Na této stránce se ponoříme do světa CSS (Cascading Style Sheets), což je zkratka pro Kaskádové styly. Představ si, že HTML je kostra tvého webu – text, obrázky, formuláře. Ale jak to všechno vypadá? Jakou to má barvu, velikost písma, rozložení? Přesně od toho je tu CSS!

CSS dodává tvým HTML prvkům styl a vizuální podobu. Bez CSS by tvé webové stránky vypadaly jako holý text bez jakéhokoliv designu. CSS ti umožní měnit barvy textu, pozadí, velikost a typ písma, zarovnání prvků, přidávat stíny, zaoblovat rohy a v podstatě cokoliv, co souvisí s vizuální prezentací.

CSS pracuje na principu pravidel. Vybereš si nějaký HTML prvek (například odstavec, nadpis, nebo tlačítko) a pak mu přiřadíš různé stylovací vlastnosti. Například: "všechny nadpisy první úrovně budou modré a vycentrované", nebo "všechny odstavce budou mít velikost písma 16px".

Tento odstavec je ostylován pomocí CSS! Klikni na mě!

Když klikneš, JavaScript změní některé CSS vlastnosti!

Pomocí CSS můžeš své webové projekty udělat krásnými a atraktivními! A přesně to se ti bude hodit pro tvé moderní foto editory a dynamické grafy. Je to jako LED světla na vánočním stromku, která rozsvítí celý design!

HTML Struktura / Kostra

Ahoj Jiřík! Dnes se podíváme na základní strukturu každé HTML stránky, její "kostru". Bez této základní struktury by webový prohlížeč nevěděl, jak tvůj kód interpretovat a zobrazit. Je to jako plánek pro stavbu domu nebo schematický diagram Enterprise!

Základní prvky HTML kostry:

Každá HTML stránka začíná a končí několika klíčovými značkami, které definují její základní části:

  • <!DOCTYPE html>: Tohle není tak úplně HTML značka, ale deklarace, která prohlížeči říká, že se jedná o dokument HTML5. Vždycky ji dej na úplný začátek!
  • <html lang="cs"></html>: Toto je kořenový element celé HTML stránky. Všechno ostatní, co patří k tvému webu, bude uvnitř těchto značek. Atribut lang="cs" říká prohlížeči, že jazyk obsahu stránky je čeština.
  • <head></head>: V této části je schovaná "hlava" dokumentu. Obsahuje meta informace o stránce, které se uživateli přímo nezobrazují. Jsou tu věci jako:
    • <meta charset="UTF-8">: Určuje kódování znaků, aby se správně zobrazovaly české znaky (diakritika).
    • <meta name="viewport" content="width=device-width, initial-scale=1.0">: Klíčové pro responzivní design, zajišťuje, že se stránka správně zobrazí na mobilních zařízeních a tabletech.
    • <title>Můj Super Web</title>: Text, který se zobrazí v záhlaví okna prohlížeče nebo na kartě.
    • Odkazy na externí CSS soubory (<link rel="stylesheet" href="styl.css">) nebo interní CSS styly (<style>...</style>).
    • Odkazy na JavaScript soubory nebo interní skripty.
  • <body></body>: Tady je to hlavní! Všechno, co se uživateli zobrazí na stránce – text, obrázky, videa, tlačítka, formuláře – to vše patří sem. Představ si to jako "tělo" tvého webu.

Takhle vypadá minimální, ale kompletní HTML kostra:


<!DOCTYPE html>
<html lang="cs">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Moje První HTML Stránka</title>
</head>
<body>
    <!-- Zde bude veškerý viditelný obsah tvé stránky, Jiřík! -->
    <h1>Vítejte na mé stránce!</h1>
    <p>Toto je ukázka základní HTML struktury.</p>
</body>
</html>

Pochopení této struktury je naprosto zásadní pro každý tvůj webový projekt, od osobní stránky až po Star Trek hudební přehrávač!

Co je to JavaScript?

Ahoj Jiřík! Jestliže HTML je kostra a CSS je vzhled, pak JavaScript je mozek a svaly tvého webu! Je to programovací jazyk, který dělá webové stránky interaktivními a dynamickými. Bez něj by tvoje stránky byly statické jako papírové noviny, nebo jako vesmírná loď bez pohonných jednotek!

JavaScript ti umožňuje reagovat na akce uživatele – například když klikne na tlačítko, vyplní formulář, nebo se pohybuje myší. Díky JavaScriptu můžeš:

  • Měnit obsah HTML elementů v reálném čase.
  • Měnit styly HTML elementů (např. barvu, velikost) bez načítání celé stránky.
  • Animovat prvky na stránce.
  • Ovládat audio a video.
  • Validovat formuláře (kontrolovat, jestli uživatel zadal správné údaje).
  • Komunikovat se serverem bez nutnosti přenačítání stránky (AJAX).
  • Vytvářet hry, jako bys chtěl pro své Star Trek projekty!

JavaScript se spouští přímo v prohlížeči uživatele, což ho dělá velmi rychlým a efektivním pro interaktivní prvky. Je to esence pro tvůj futuristický cestovatelský kvíz nebo rozšířený HTML editor!

Klikni na tlačítko a uvidíš kouzlo JavaScriptu!

Přípojný soubor <script>

Ahoj Jiřík! Už víš, že JavaScript dělá web interaktivním. Ale jak se ten JavaScript kód vlastně dostane do tvé HTML stránky? K tomu slouží značka <script></script>.

Kde a jak použít <script>?

Značka <script> může být použita dvěma hlavními způsoby:

  1. Vložený JavaScript (Internal JavaScript): Kód JavaScriptu napíšeš přímo mezi otevírací a uzavírací značku <script>.

    
<script>
    // Tento JavaScript kód je přímo v HTML dokumentu
    console.log("Ahoj, Jiřík! Toto je inline JavaScript.");
</script>

    Tento způsob je vhodný pro menší skripty, které jsou specifické pro jednu stránku.

  2. Externí JavaScript (External JavaScript): Tohle je nejpoužívanější a nejlepší praxe! JavaScript kód uložíš do samostatného souboru (např. script.js) a na něj pak odkážeš pomocí atributu src v značce <script>.

    
<script src="muj-script.js"></script>

    Výhody externích souborů:

    • Čistota kódu: HTML, CSS a JavaScript máš oddělené, což je přehlednější.
    • Znovupoužitelnost: Jeden JS soubor můžeš použít na více stránkách.
    • Rychlejší načítání: Prohlížeč může externí JS soubor uložit do mezipaměti (cache), takže se nemusí při každém načtení stránky stahovat znovu.

Kam umístit značku <script>?

Můžeš ji umístit buď do sekce <head> nebo do sekce <body>. Nejčastější a doporučené místo je těsně před uzavírací značku </body>. Proč?

  • Když prohlížeč narazí na <script> značku, pozastaví parsování HTML a nejprve stáhne a spustí JavaScript. Pokud je JavaScript v <head> a je velký, může to zpomalit zobrazení obsahu stránky.
  • Pokud je JavaScript na konci <body>, HTML obsah se nejprve načte a zobrazí uživateli, a teprve pak se spustí skripty. Tím zajistíš, že uživatel uvidí něco hned, i když se JavaScript ještě načítá. Také zajistíš, že JavaScript bude mít přístup ke všem HTML prvkům, které chce manipulovat, protože už budou "načtené".

Tip pro pokročilé, Jiřík: Existují také atributy async a defer, které můžeš přidat ke značce <script src="...">. Tyto atributy mění způsob, jakým prohlížeč stahuje a spouští externí JavaScript soubory, a mohou výrazně zlepšit výkon stránky!

  • async: Stahuje skript asynchronně (na pozadí) a spustí ho, jakmile je stažen, aniž by blokoval parsování HTML.
  • defer: Stahuje skript asynchronně, ale spustí ho až poté, co je celý HTML dokument naparsován a DOM (Document Object Model) je připraven. To je obvykle nejlepší volba pro skripty, které manipulují s HTML prvky.
  • Používám defer pro Tailwind CSS skript v hlavičce, aby se načítal na pozadí a neblokoval vykreslování stránky.

Nyní už jsi připraven kombinovat tyto znalosti a tvořit kompletní a funkční webové aplikace!

Kompletní HTML Kód pro Jiříka

Ahoj Jiřík! Gratuluji! Dostali jsme se až sem. Na této stránce uvidíš, jak se všechny dříve probrané koncepty – HTML struktura, CSS styly a JavaScript logika – spojí dohromady v jeden funkční a interaktivní celek.

Tento kód ti ukazuje typickou strukturu webové stránky. Má základní HTML kostru, obsahuje interní CSS pro vizuální design a používá interní JavaScript pro interaktivitu. Můžeš to brát jako startovní bod pro tvé budoucí skvělé projekty, jako je třeba ten Star Trek hudební přehrávač, kde bys mohl nechat blikat LED světla do rytmu hudby!

Ukázka kompletního kódu:


<!DOCTYPE html>
<html lang="cs">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Můj interaktivní web</title>
    <!-- Odkaz na externí styly (např. Tailwind CSS nebo tvůj vlastní soubor style.css) -->
    <link href="https://cdn.jsdelivr.net/npm/tailwindcss@2.2.19/dist/tailwind.min.css" rel="stylesheet">
    <style>
        /* Interní CSS styly - pro menší projekty nebo specifické úpravy */
        body {
            font-family: Arial, sans-serif;
            background-color: #f0f0f0;
            color: #333;
            text-align: center;
        }
        .my-button {
            background-color: #4CAF50; /* Zelená */
            color: white;
            padding: 15px 32px;
            text-align: center;
            text-decoration: none;
            display: inline-block;
            font-size: 16px;
            margin: 4px 2px;
            cursor: pointer;
            border-radius: 8px;
        }
    </style>
</head>
<body>
    <h1>Vítejte na mém interaktivním webu!</h1>
    <p id="myParagraph">Tento text se změní po kliknutí na tlačítko.</p>
    <button class="my-button" onclick="changeParagraphText()">Klikni na mě!</button>

    <!-- Externí JavaScript soubor -->
    <script src="muj-interaktivni-script.js"></script>

    <!-- Interní JavaScript (nebo po načtení externího, pokud na něm závisí) -->
    <script>
        function changeParagraphText() {
            document.getElementById('myParagraph').textContent = 'Text byl změněn JavaScriptem! Super, Jiřík!';
            document.getElementById('myParagraph').style.color = '#e74c3c'; // Změna barvy CSS pomocí JS
        }
        // Ukázka interakce s uživatelem a generování obsahu
        // Zde bys mohl vložit kód pro tvůj futuristický kvíz nebo editor
    </script>
</body>
</html>

Všimni si, jak jsou jednotlivé části propojeny. HTML vytváří strukturu (nadpis, odstavec, tlačítko), CSS (a Tailwind) se stará o jejich vzhled (barvy, velikosti, zaoblení) a JavaScript dodává interaktivitu (mění text a barvu po kliknutí). Tady bys mohl začít s implementací tvých nápadů na Moderní foto editor nebo Star Trek kapitoly!
